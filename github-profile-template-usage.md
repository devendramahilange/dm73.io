# GitHub Profile Template Usage

This output contains a ready-to-use GitHub profile template based on the screenshot.

## Use It

1. Create a GitHub repository named exactly like your GitHub username.
2. Rename `github-profile-template.md` to `README.md`.
3. Put it in that repository.
4. Replace `Your Name`, `Alex Johnson`, job title, company, links, project names, and `yourusername`.
5. Replace the profile image URL with your own GitHub avatar URL.
6. Commit and push.

GitHub profile READMEs do not allow full custom CSS, so this version uses GitHub-safe HTML, tables, badges, and images.

## Optional CV

The template links to:

```text
./resume.pdf
```

Add your CV with that file name, or update the link in the template.

## Rich Portfolio Version

Use `github-pages-profile-template.html` for GitHub Pages when you want the custom card layout, gradient background, and education cards from the screenshot.
